
# VIM commands:

### Navigation:
| Command        | Use           | Note  |
| ------------- |:-------------:| -----:|
| **w**   | Move start of next word |  |
| **e**   | Move end of next word   |    |
| **b**   | Move start of back word      |     |
| **Shift + %** | Move start & End of {([])} | good for programming|
| **Shift + {**| Previous para ||
| **Shift + }**| Next para | |

| ****| | |
